A BIOS (Basic Input Output System, ou Sistema Básico de Entrada de Saída) é uma firmware, uma interface principal entre os hardware e o sistema operacional e é responsável por iniciar o computador. O SETUP é um programa responsável pelas configurações de alguns hardwares conectados à placa mãe. o POST (Power-On Self-Test, ou Auto-teste de Inicialização) é uma sequência de carregamentos que indica se o computador está apto a inicialização. Por fim, o boot, é o processo de ligar o computador que inicia o Sistema Operacional.
