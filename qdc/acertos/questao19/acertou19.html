O Setup contém todas as informações para que o sistema reconheça os componentes instalados no computador: se qualquer dispositivo não for identificado ou localizado pelo BIOS, você terá problemas para fazê-lo funcionar no sistema operacional. As informações da identificação e localização dos componentes ficam gravadas no CMOS (Complementay Metal Oxide Semicondutor), que é uma memória RAM (volátil) usada para gravar as configurações do Setup da placa mãe, embutida no chip do BIOS.
